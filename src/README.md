# Employee Management Dashboard

## Overview
A React-based Employee Management Dashboard that provides authentication and core employee management features such as creating, updating, deleting, searching, and filtering employee records.

## Tech Stack
- React.js
- Vite
- JavaScript
- Local Storage

## Features
- Mock login authentication with protected dashboard access
- Add, edit, and delete employee records
- Profile image upload with preview support
- Search employees by name
- Filter employees by gender and active/inactive status
- Print employee list
- Clean, responsive, and user-friendly UI

## How to Run
1. Clone or unzip the project
2. Navigate into the project folder
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Open http://localhost:5173 in your browser

## Future Improvements
The application is designed to be modular and extensible. Features such as backend integration, advanced filtering, role-based access, and server-side persistence can be added in future iterations based on business requirements.
